# 🤖 AI News Pilot (Note9 Edition)

這是一個由 **Samsung Galaxy Note9** 手機全手動部署的 AI 新聞自動化系統。

## 🚀 運作流程
1. **Fetch**: 使用 NewsAPI 抓取最新的 AI 科技新聞。
2. **Process**: 透過 Google Gemini 1.5 Flash 進行智慧摘要。
3. **Notify**: 自動寄送精簡日報至指定的 Gmail。

## 🛠️ 技術棧
- **Hardware**: Samsung Galaxy Note9 (The Legend)
- **Language**: Python 3.10
- **Automation**: GitHub Actions
- **AI Model**: Google Gemini 1.5 Flash
- **API**: NewsAPI

## 📅 自動化設定
系統預設台灣時間每天早上 8 點 (UTC 00:00) 自動執行。
